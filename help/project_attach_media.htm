Project Attach Media

 Top  Previous  Next

The Project Attach Media process allows the user to attach Media and/or related Media Contents directly to one or more tasks in the project workflow section.

 

-Attaching a Media allows a task under the Track Project section to be tracked on the Media level.

 

-Attaching a Media and one or more Media Contents allows a task to be tracked under the Track Project section on the Media level, and allows the task to dynamically track subtasks.  The subtasks listed will be the Media Contents (Custodians).

 

This feature is directly linked to the Media module and will only display a list of Media and/or Media Contents that has been associated to the Case during the process add/edit new Media Contents process.

 

 

-Select one or more tasks in the project workflow section

 

-Select the icon to display a list of Media and Media Contents that is associated with the active case the user is on

 

 

-Select one or more Media headers to only attach a Media

 

-Select one or more Media Contents rows to attach a Media and Media Contents (Custodians)

 

-Select Attach button

 

The project workflow task will now change to reflect the attached Media and/or Media Contents

 

 

-Select the icon next to a Media and/or Media Contents name to remove from a task.
