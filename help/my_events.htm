The My Events section displays projects via their duedate in a calendar view.

The user can select any day within the calendar to launch an expanded view of the calendar with all of the projects associated with the users cases or teams:

The user can click once on any project listed on the day to view it high level project details on the right:

 

The user can double-click on any project listed on a day to redirect the user to the actual project details in the my cases or my teams module.

The user can select Month, Week, or Day to view project in the selected display:

The user can select My Teams or My Cases button to view all projects in the case or team:

The user can select My Submitted button to view only those projects the user submitted.
