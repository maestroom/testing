Subscribe To Email Alerts

 Top  Previous  Next

 

Each user has the ability to subscribe to email alerts based on various activities performed throughout the application.

If an Alert Force option is applied to a Role or User Security setting, then the user may not be allowed to edit an subscription setting.  The user will be forced to receive an alert.

-Select the Options icon

-Select Subscribe to Email Alerts

-Select preferred alerts

-Select Update
