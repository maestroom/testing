Track Case Project

 Top  Previous  Next

The Track Case Project submodule is where users track the progress of the actual project tasks.  A project consists of many tasks that are a part of a defined workflow that is created and submitted under the Submit New Project process.  If a user performs a Change Project process under the My Cases module, all of the changes to the workflow and workflow instructions will be dynamically updated in the Track Project submodule.

 

 

There are two ways to enter the Track Project submodule from the Case Project submodule:

 

1.        Select the icon in the Status field in the grid display

 

2.        Select a project in the grid display and select Track Project node in the left section
