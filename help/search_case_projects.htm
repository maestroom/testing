Search Case Projects

 Top  Previous  Next

The Search Case Projects feature allow the user to select one or more active cases and perform keyword searches against all text added to project instructions, comments, todos data.

 

 

-Select one or more active cases in the grid display

 

-Enter text in the enter search box

 

-Select Search button

 

The search results will appear at the bottom of the screen where the user can review and select a result link to navigate directly to the project in the case portal:
