Today's Activity

 Top  Previous  Next

 

The Today's Activity section list specific activities users perform across the application regarding the cases and teams the user has permission to.

This contents of the list display the last 25 entries, with the ability to see more, and only displays the past 24 hours of activity.  Each day the list refreshes to see only the activity in the current day.

The Today's Activity window can be disabled from view per the user's permission rights.
