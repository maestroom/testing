Edit Operation Team Service Tasks

 Top  Previous  Next

-To edit Services Tasks, Select Edit button under the Service Tasks tab:

-Edit Task Name.

-Edit Description (Optional).

-Edit Is Billable Item:
 

-Selecting Yes will add this task to the Billing Module under Price Point section.

-Selecting Yes will add the Billable Items section under the Track Project section to add billing units to a Task.

-Edit Is Billable Item with Force Entry:
 

-Selecting this option will force a user to add a billable item before the task can be flagged as "Complete"

-Edit Hide option:
 

-Select Hide if you want to hide the Task from view.  The Task will still remain in backend table so all information is still linked to projects.

-Select Update button.
