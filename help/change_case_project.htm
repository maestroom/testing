Change Case Project

 Top  Previous  Next

The Change Case Project section allows the user to change the project after it has been submitted.  Project can only be changed in the My Cases module.

 

 

-Select the My Cases module and enter a case portal

 

-In the Case Projects submodule, select a project in the grid display

 

-Select the Change Project node in the left section

 

 

Under the Change Project section:

 

-Select Change Project button

 

 

-Edit or add additional project instructions

 

-Select the Next button to work through the forms and select Resubmit button at the end of the form process to resubmit the project with all of the revised instructions.

 

-Each Change Project process will be version controlled

 

 

-Click on a version number to view the exact changes highlighted in tan color

 

 

-The changes will also be reflected in the My Cases and/or My Teams Track Project section when a user views the project instructions
