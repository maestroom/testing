My Cases Statistics

 Top  Previous  Next

 

The My Cases Statistics areas displays the following project statistics grouped on the case level:

 

 

-Grid Statistics:

 

 -Active Projects:  displays the count of all active projects in the case portal

 

 -Added ToDos: displays the count of all active ToDos in the case portal

 

 -Unread Comments: displays the count of all unread comments in the case portal

 

         -The user has the ability to view all of the comments at once in addition to filtering only the projects that have the comments:

 

         -Select the icon next to the comment value in the grid to display the comments in the below frame to review in bulk.  Note the color of the icon changes to red:

 

         \

 

         -Select the Clear button to remove the comments display after reading them.  After the Clear button is selected, the unread comments number value will return to 0.

 

         

 -Unassigned Projects: displays the count of all projects that have been unassigned to any user

 

-Project Status chart:

 

 -Displays the activity status of all active projects.

 

-Project Priority chart:

 

 -Displays the priority status across all active projects

 

Note:  All of these sections may be selected to filter the appropriate grid and chart display information in the case portal projects submodule
