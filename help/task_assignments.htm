The Task Assignments submodule to view what and how many active tasks users are assigned:

-Select Task Assignments submodule

-Select one or more teams to view all active assignments.  Active means tasks and todos that have yet to be completed.

-Select a specific date to filter all assignments on a specific day

-Select PDF to export statistics

-Select Clear All to clear the dynamic statistics window

-Click on a user name to view the assignment breakdown

-Hyperlink any of the blue values to launch the actual task in Track Project section

-Select the PDF button to export the statistics

-Select Back button to redirect back to the resource page
