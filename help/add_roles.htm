-Select User Roles node

-Enter User Role details
 

-Note there are two types of Role Types to select:
 

-Case Manager - select this option if all users assigned to this role will manage cases and have access to the My Cases module

-Team Member - select this option if all users assigned to this role will work within teams that receive and work on project tasks and have access to the My Teams module

Note: A role can be assigned both role types or one or the other

-Select Next button

-Select the specific features throughout the application that all users assigned to this role will have access to

-Select Add button

Note: The Options - Subscribe to Email Alerts security section as this section differs in selection activity:

 

-Select an Alert option only to allow the user to see the alert type under Options section.  User still has the option to subscribe to the alert:

-Select both the Alert option and the Force option to allow the user to see the alert type under Options section.  User will be forced to receive the alert and does not have the option to select/deselect alert option:

-Seleting no Alert option and user will not see the alert option and will not be allowed to subscribe to the alert:
