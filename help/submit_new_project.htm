Submit New Case Project

 Top  Previous  Next

The Submit New Project process can only be achieved in the My Cases module under a specific Case portal.  Every new project that is submitted will be automatically linked to the Client and Case it was submitted under:

-Select the My Cases portal

-Select the Case portal the project needs to be submitted under

-After you enter the Case portal, select the New Project button

-Enter project Deadline details

-Select Next button

-Select any combination of Available Service Tasks in the list to build a custom project workflow process

Note: Instead of building a custom project workflow process, the user can select the Load Prev Workflow button to carry over a previous projects workflow and prior entered instructions.

-Select the icon to move the selected task to the Project Workflow section

 -The Project Workflow section is designed to build custom project workflows that determines the tasks that will be worked on across all teams, and        determines the flow in which tasks will be worked on across all teams.

 -The Tasks added to the project workflow section can be dragged and dropped to allow for a specific order of the project

 -Specific Tasks can be removed from the workflow section by selecting the task(s) and selecting the icon

 -A user can Attach Media and associated Media Custodians can be attached to each task if the user wants to track tasks at the Media level

 -A user can add Estimated Time attributes to each Task if the user want to track the Estimated time of a project against the Projected time and Actual Time

-Select Next button

-Enter Form Details for each Task that were selected in the project workflow.

-Select the icon in each of the Task section headers to attach additional instruction documents that relate to the task instructions

-Select Save button to save the project prior to Submitting the project.  This is a good idea if the user cannot Submit the project because the user needed to obtain additional instructions from the client.  The user can select the Saved Projects button in Case Projects area to relaunch the project workflow to finalize the instructions to submit.

-Select Submit button to finalize the submission process and the project is now live in the system and will appear in the Case Project Grid display
