Add Media Contents

-Select the Add Contents button

-Select Associated Cases Media media is associted with

-Select Next button

-Enter Media Contents details

Each Custodian added to the system has a unique ID associated with it and will appear in the dropdown list if it was already added to the system.  Check the dropdown list first to see if the custodian name exists:

-If a Custdian does not appear in the dropdown list, Select Add New button:

-Enter the Custodian name details

-Select Add

The Custodian name will now appear in the Select Custodian dropdown list

After all details have been added to the Media Contents item,

-Select Add button to only enter one media contents and be redirected back to main media contents grid list

-Select Add Another button to enter multiple media contents quickly by redirecting the user back to the enter media contents form screen
