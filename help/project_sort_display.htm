Project Sort Display

The Project Sort Display node is where specific project grids throughout the application are sorted by default.  The option selected affects all users in the application as they navigate to the affected grids:

The following grids will be sorted automatically for all users:

1. My Cases and My Teams Assignments tabs located in the My Assignments module

 

2. Case Projects grid located in the My Cases module

3. Team Projects grid located in the My Teams module
