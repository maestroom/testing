Case Project Grid

 Top  Previous  Next

The Case Project Grid lists all of the projects that are submitted in the case.

 

The user may select any row in the grid and perform any action on the left hand side.  The user may also click on highlighted field contents and icons to auto-launch into specific areas of the case portal instead of selecting the row and using the nodes on the left side.
