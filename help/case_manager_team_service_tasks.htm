Case Manager Services Tasks - A Service Task is an activity that needs to be accomplished within a defined period of time or by a deadline.  Tasks are measurable units that can be assigned, transitioned, started, paused, on-hold and completed.   One or more tasks can be added and it all depends on the organization and how deep they want to measure the work within the Team Service.  Any combination of tasks may be added to a custom project workflow.

-The service task grid rows may be dragged and dropped to custom sort the task display.

-The Form? grid field indicates if a form template has been created or created/published.

- means the form has been created and is in save mode and not available to public use

-means the form is published and available for case manager to access under New Project process

-The Billable? grid field indicates if a task has been categorized as a billable task or billable task with force entry:

- means the task is a billable task with force entry option selected. Force option means the user has to add a billable item before the task can be marked "complete"

-means the task is a billable task without the force entry option selected
