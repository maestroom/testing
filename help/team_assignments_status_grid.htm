Team Assignments Status Grid

My Teams Assignments tab indicates the logged in user's assignments that relate to a Operation team tasks.

-The grid is organized by teams.

-The Tasks column lists the number of tasks that have been assigned to the user

-The Pending column lists the number of tasks that are ready to work on per the project workflow order

-The ToDos column lists the number of todos that have been assigned to the user

-Each of the three columns link and filter the assignments in the My Teams portal when the values are selected

When the icon next to the Project Name is selected in the Project Name column, the below charts are dynamically changed:

-Project Task Status - displays the entire project status across all tasks

-Project Task Assignments - displays the entire project assignments across all users

-Project ToDo Assignments - displays the entire todo assignments across all users

The My Teams Assignments tab also directly feeds the My Events area, meaning that when the My Teams Assignments tab is selected, and a day is selected in the My Events calendar, the calendar displays projects that are displayed are projects appearing your assigned teams.
