Edit Form to Case Manager Team Service Tasks

 Top  Previous  Next

-To edit Task Form, Select a Task row in the grid under the Service Tasks tab.

-Select the Edit Form button:

-Select Edit icon to edit elements after it has been dynamically added to the form template:

Note: It is recommended that you do not use any special characters in any of the fields in the Edit form process.

-Text Heading:

       

-Text Mult, Text Single & Date Picker:

 

         

 -Label - Name of form field

 -Required - option to indicate form field is required before submitting a new project process

 -No Load Previous - option to not transfer any applied data during the load previous project process

 -Description - Add a brief description to explain the form field

-Dropdown, Checkbox & Radio Button:

 

         

-Label - Name of form field

-Required - option to indicate form field is required before submitting a new project process

-No Load Previous - option to not transfer any applied data during the load previous project process

-Description - Add a brief description to explain the form field

-Options - Add one or more entries for selection.  Separate each criteria by a ; for mulitple entries.

-Select Preview button to test form prior to Publish

-Select Save button to save for prior to Publish

-Select Publish to accept all changes and modifications are active in the New Project process
