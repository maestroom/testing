The My Cases Portal is a module where all project activity is organized and managed at the case level.

 

To enter a Case Portal users can perform the following:

 

-Select a Case in the dropdown list and select Enter button.  The dropdown list will only contain cases the user has access to under the Manage Users section.

 

-Select any case name or values in the grid display or any of the chart bars to filter the appropriate projects in the portal's Case Projects submodule.

 

Note: Only Cases with active projects (active means only those projects that are not complete), appear in the grid and chart sections.  To enter a case portal that has newly created or a case that does not have Active project needs to be selected in the dropdown list along with the Enter button process.  

 

 

 

-After any selection has been selected, the user will be redirected to the selected Case Portal:

 

 

 

-Case Projects

 

-Case Budget

 

-Case Documents

 

-Case Reports
