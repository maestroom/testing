User Security

 Top  Previous 

The User Security tab is part of the Edit User process.  This tab allows the user to have additional feature rights that are above and beyond the rights the associated Role has.  This is designed to give one or more additional rights to a user where the Role they are associated with does not have.  Adding one or more additional right prevent the user from switching to another Role or the need to create another Role just to allow this one user added feature right.

-Select User node

-Select User Security tab

-Select or remove additional feature rights above and beyond the Role rights the user is associated with

-Select Update button
