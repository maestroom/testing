Submit Saved Case Projects

 Top  Previous  Next

The Submit Saved Case Projects feature allows the user to retrieve previously saved projects to continue entering project instructions in order to submit the project:

 

 

-Enter My Cases portal and enter a case portal

 

-Under the Case Projects submodule, select the Saved Projects button

 

 

Under the Saved Project section:

 

-Select a Saved Project in the grid display

 

-Select the Edit Saved Project node

 

 

-Edit or add additional project instructions

 

-Select the Next button to work through the forms and select Save again at the end if the user is not ready to Submit the project or select Submit button to finalize the project.

 

-Delete Saved Projects
