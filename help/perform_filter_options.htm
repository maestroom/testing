Perform Filter Options

 Top  Previous  Next

 

-Select Saved Filters submodule

-Select one filter at a time on left side to retrieve the selected filter results

-Select Filter Options button:

-Select one option in the left section

-Bulk Close Projects - selecting this option will apply a close status for all projects appearing in the filter list

-Bulk ReOpen Projects - selecting this option will apply a reopen status for all projects appearing in the filter list

Note: These bulk operations are applied across the selected filter list.  The user cannot select only a few rows to use the bulk operations.  The action will only be applied across entire filter list.
