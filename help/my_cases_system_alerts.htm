My Cases System Alerts

 Top  Previous  Next

The My Cases System Alerts notifies the user that the following activity has occurred in one of the cases the user has access to:

 

-The system alerts the user when a new project has been submitted

 

-The system alerts the user when a new ToDo has been added

 

-The system alert the user when a new comment has been posted

 

 

These system alerts are auto-generated and the user does not have to be in the exact module where the alert is derived from.

 

In order the for the user to see which case the new item has been added to:

 

 

-When the (New) flag appears in the specified module, select the My Cases (New) tab to display the case highlighted where the activity had occurred:
