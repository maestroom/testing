My Cases

 Top  Previous  Next

 

The My Cases module organizes projects by client cases.  Each case has its own unique portal where all projects are created and managed at the case level.

 

Only users with Role Type equal to Case Manager will have access to this module.

 

Uses with specific User level security rights will only view the cases they have access to.
